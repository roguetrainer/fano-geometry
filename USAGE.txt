FANO GEOMETRY MODULE 1 - USAGE GUIDE
=====================================

AFTER UNZIPPING
---------------

1. Navigate to the directory:
   cd fano-geometry-module1

2. Install dependencies:
   pip install -r requirements.txt

   OR install as a package:
   pip install -e .


WAYS TO USE THE CODE
--------------------

METHOD 1: Run the demo scripts
-------------------------------
cd src
python fano_geometry.py        # See structure and verify axioms
python fano_visualization.py   # Generate visualization images


METHOD 2: Interactive Python
-----------------------------
cd src
python

>>> from fano_geometry import FanoPlane, verify_axioms
>>> from fano_visualization import FanoVisualizer
>>> 
>>> # Create Fano plane
>>> fano = FanoPlane()
>>> 
>>> # Basic queries
>>> fano.line_through_points(0, 1)  # Returns: 0
>>> fano.intersection(0, 1)          # Returns: 1
>>> fano.is_collinear(0, 1, 3)      # Returns: True
>>> 
>>> # Visualize
>>> import matplotlib.pyplot as plt
>>> viz = FanoVisualizer(fano)
>>> viz.plot_standard()
>>> plt.show()
>>> 
>>> # Verify axioms
>>> verify_axioms(fano)


METHOD 3: Jupyter Notebook (RECOMMENDED!)
------------------------------------------
cd notebooks
jupyter notebook 01_basic_fano.ipynb

This opens an interactive tutorial that walks you through:
- Fano plane construction
- Axiom verification
- Multiple visualizations
- Structural exploration
- Symmetry analysis
- Connection to coding theory


METHOD 4: Run tests

cd tests

All tests should pass with green checkmarks!

QUICK EXAMPLES
--------------

Example 1: Find line through two points
----------------------------------------
from fano_geometry import FanoPlane
fano = FanoPlane()
line = fano.line_through_points(2, 5)
print(f"Line through 2 and 5: {line}")
print(f"Points on line: {sorted(fano.points_on_line(line))}")


Example 2: Check collinearity
------------------------------
from fano_geometry import FanoPlane
fano = FanoPlane()
print(f"0,1,3 collinear? {fano.is_collinear(0,1,3)}")  # True
print(f"0,1,2 collinear? {fano.is_collinear(0,1,2)}")  # False


Example 3: Visualize
--------------------
from fano_geometry import FanoPlane
from fano_visualization import FanoVisualizer
import matplotlib.pyplot as plt

fano = FanoPlane()
viz = FanoVisualizer(fano)

# Show all representations
viz.plot_all_representations()
plt.show()


Example 4: Incidence matrix
----------------------------
from fano_geometry import FanoPlane
fano = FanoPlane()
M = fano.incidence_matrix()
print("Incidence matrix:")
print(M)
print(f"\nEach point on {M.sum(axis=1)[0]} lines")
print(f"Each line has {M.sum(axis=0)[0]} points")


Example 5: Cyclic symmetries
-----------------------------
from fano_geometry import FanoPlane
fano = FanoPlane()

# Rotate by 3 (mod 7)
auto = fano.automorphism_cyclic(3)
print(f"Point 0 maps to: {auto[0]}")
print(f"Point 1 maps to: {auto[1]}")
print(f"Automorphism: {auto}")


TROUBLESHOOTING
---------------

"No module named 'fano_geometry'":
  - Make sure you're in the src/ directory, OR
  - Install the package: pip install -e . (from root), OR
  - Add to Python path: sys.path.insert(0, 'path/to/src')

"ModuleNotFoundError: No module named 'numpy'":
  - Install dependencies: pip install -r requirements.txt

Jupyter notebook won't open:
  - Install jupyter: pip install jupyter
  - Or: pip install -r requirements.txt

Tests fail:
  - Check that you're in tests/ directory when running
  - Ensure all dependencies installed
